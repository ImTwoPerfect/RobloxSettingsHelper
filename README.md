# RobloxSettingsHelper
**RobloxSettingsHelper** is a tool availiable to Windows users to unlock Roblox FPS, change textures, fonts, and automatically update ClientAppSettings without needing a Bootstrapper (i.e. Bloxstrap)

To install, simply extract the files and run RobloxSettingsHelper.exe. If you don't change any of the files, all of these effects will be applied to your Roblox. If you don't want dark textures, you can delete the textures folder inside. If you don't want your font to change, you can delete font.ttf inside. If you don't want your ClientAppSettings.json to change, you can delete the ClientSettings folder inside.
